# Community findings

Independent researchers can **publish additional results** here.

## How to add yours

1. Read **[CONTRIBUTING.md](../../CONTRIBUTING.md)**  
2. Copy [`TEMPLATE_FINDING.md`](TEMPLATE_FINDING.md) into a new folder:

```text
community/findings/YYYY-MM-DD_short-title/README.md
```

3. Add evidence under `evidence/` if needed  
4. Open a **Pull Request** on GitHub

## Scope

- Reverse engineering of the combo ESP32 firmware image  
- Hardware variants and measurements  
- Improvements / corrections to rebuild packages  
- Educational RF / IEEE notes (authorized lab context only)  

**Not for** operational jamming guides or attacks on third parties. See repo [DISCLAIMER.md](../../DISCLAIMER.md).
